# 모듈러 운송 시뮬레이터 참고 자료

본 시뮬레이터에서 사용된 외부 자료 모음. 모든 수치·가정값의 출처를 명시.

## ⚠ 중요한 사실

본 시뮬레이터의 차량 카탈로그는 **모듈러 건축 전용 운송 차량**으로만 구성:
- 일반 카고 트럭은 적재함 폭 2.35m라 모듈 운송 **불가**
- 모듈은 광폭 운송 허가를 받은 **저상 트레일러** 또는 **A-frame 트레일러** 사용

## 출처 신뢰도 단계

- ★★★ — 공식 정부·기관 발간 (LH 매뉴얼 §2.7.1, KOSHA 가이드)
- ★★ — 산업협회·전문 단체 (PCI MNL-122, Wells Concrete, 한국특장차)
- ★ — 운송업계 일반 정보 (forwarder.kr, 운송사 사양표)

학부 종합설계 발표용으로는 ★★ 이상 자료 위주로 작성됨. 실제 본 조 발표·설계서 인용 시 1차 자료(LH 원문 PDF, PCI MNL-122 직접 다운로드) 직접 받아 검증 권장.

## 한계 및 보완 필요

1. **한국특장차·동우특장차 등의 정확한 모델별 제원**은 메이커 직접 문의 필요 (kortrailer.com 사양표 미공개)
2. **광폭 운송 허가 절차의 실제 비용·기간**은 운송사 견적서 확인 필요
3. **PCI MNL-122 원문 PDF**는 로그인 후 다운로드 필요 — 본 자료는 발췌·요약본
4. **본 조 보유 자료** (GS건설 옥탑·공릉동 SH·포스코 광양생활관)에 운송 차량 정보가 있을 가능성 — 추가 확인 권장
