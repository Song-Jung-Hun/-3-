## 단위중량 산출식

```
단위중량 (kg/m) = 단면적 (mm²) × 밀도 (kg/m³) × 1m / 10⁹
                = 단면적 × 7,850 / 10⁹
                = 단면적 × 7.85 × 10⁻⁶  [kg/mm·m × m]
```

**SHS 단면적**:
```
A = w × h - (w - 2t) × (h - 2t)
```
**H형강 단면적**:
```
A = 2 × bf × tf + (h - 2tf) × tw
```

## 본 시뮬레이터에서의 사용

### 모듈 무게 산출
```
모듈 = 4 모서리 기둥 + 4 천장보 + 4 바닥보 (총 12 부재)

무게 = 4 × 모듈높이 × 기둥 단위중량(kg/m)
     + 4 × (모듈폭 + 모듈길이) × 보 단위중량(kg/m) × 2(천장+바닥)
```

**예**: 침실 모듈 3.0×6.0×3.0m, 기둥 SHS 200×200×8 (47 kg/m), 보 SHS 150×150×6 (26.4 kg/m)
- 기둥: 4 × 3 = 12 m × 47 = 564 kg
- 보: 4 × (3+6) × 2 = 72 m × 26.4 = 1,901 kg
- **합계: 약 2,465 kg/매** (프레임만)

### 플로어 패널 무게 산출 (콘크리트 바닥판 제외)
```
무게 = 둘레(2×(폭+길이)) × 보 단위중량
```

### 벽체 패널 무게 산출 (벽체판 제외)
```
무게 = 2×폭 × 보 단위중량 + 2×길이 × 기둥 단위중량
       (위·아래 보 2개)      (양쪽 기둥 2개)
```

### L자 패널 무게 산출 (바닥+벽 합체, 콘크리트 제외)
```
무게 = (3×폭 + 2×바닥길이) × 보 단위중량 + 2×벽높이 × 기둥 단위중량
       (바닥 먼변 + 꺾임 + 벽 윗변 = 3×폭)
       (바닥 양옆 = 2×바닥길이)
       (벽 양쪽 기둥 = 2×벽높이)
```
