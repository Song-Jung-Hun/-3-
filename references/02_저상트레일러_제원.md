## 출처
- 한국특장차 (https://kortrailer.com) "3축 초저상 슬라이드 로우베드 트레일러"
- 코머신 한국특장차 제품 페이지 (https://www.komachine.com/ko/companies/korea-trailer/)
- 동우특장차 "2축 저상형 트레일러 LBAAD-01" (https://www.komachine.com/ko/companies/dongwoo-motor/products/18697)
- 미국 Heavy Haulers 모듈러 운송 사례 (RGN/Lowboy 비교) (https://www.heavyhaulers.com/mobiles/modular_home_transport.php)
- LH 모듈러주택 업무매뉴얼 §2.7.1 도로별 허용 기준

## 신뢰도
★★ — 한국 트레일러 메이커 자료 (정확한 모델별 제원은 메이커 견적서 확인 필요)

## 저상 트레일러란

화물 적재면이 일반 평판 트레일러(900~1100mm)보다 낮은 **400~700mm**인 트레일러. 높이 한도(4.5m)를 더 잘 활용할 수 있어 큰 화물 운송에 적합. **한국 모듈러 건축의 모듈 운송에 가장 일반적으로 사용**.

## 표준 3축 저상 트레일러 제원

| 항목 | 수치 | 비고 |
|------|-----|------|
| 적재함 길이 | 12m | 모듈 1~2개 적재 가능 |
| 적재함 폭 | **3.0m** | 폭 3m 이하 모듈/광폭 패널 가능 |
| 바닥 높이 | 0.4~0.7m | 일반 카고(0.9m)보다 낮음 |
| 적재 중량 | **24t** | 광폭 운송 허가 차량 |
| 축수 | 3축 | 무게 분산 |
| 트랙터 결합 시 총 길이 | 약 16m | 도로 한도 19m 안 |

## 확장형 광폭 트레일러 (Extendable Lowbed)

광폭 모듈(3.4m)·장축 모듈(12m+) 운송용:

| 항목 | 수치 | 비고 |
|------|-----|------|
| 적재함 길이 | 16~22m (확장) | 슬라이드로 길이 조절 |
| 적재함 폭 | **3.4m** | LH 매뉴얼 광로 한도 |
| 바닥 높이 | 0.4~0.7m | |
| 적재 중량 | **30t** | 거실 광폭 모듈(9~12t) 1~2개 |
| 광폭 운송 허가 | **필수** | 국토부 제한차량 운행허가 |
| 호송차량 | 1~2대 | 야간 운송 의무 (3.4m 광폭) |

## 광폭 운송 허가 절차

1. 국토교통부 제한차량 운행허가시스템 (https://www.ospermit.go.kr/) 접속
2. 운송 경로 (출발지·목적지) 입력
3. 차량 제원 + 화물 제원 입력 (폭, 길이, 높이, 중량)
4. 검토 (3~5일 소요)
5. 허가증 발급 → 호송차량 동반 + 야간 운송

## 모듈 적재 매수 (이론)

저상 트레일러 적재함 12m × 3.0m 기준:

| 모듈 | 가능 매수 (1단) | 비고 |
|---|---|---|
| 침실 모듈 (3.0×6m) | 2매 | 길이 방향 2매 |
| 거실 모듈 (3.0×9m) | 1매 | 길이 12m 한도 |
| 수직 모듈 (2.4×3m) | 4매 | 1단 4매 가능 |

확장형 광폭 22m × 3.4m: 거실 광폭 모듈(3.4×9m) 2매 가능.
