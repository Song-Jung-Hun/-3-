## 한국 도로 운행 기본 한도 (도로교통법 시행규칙)

| 항목 | 일반 운행 | 광폭 운송 허가 시 |
|------|---------|---------------------|
| 차량 폭 | **≤ 2.5m** | 2.98~3.64m (허가 필수) |
| 차량 높이 | ≤ 4.0m | ≤ 4.5m |
| 트랙터+트레일러 합산 길이 | ≤ 16.7m | 평판 등 ≤ **19m** |
| 축중 한도 | ≤ 10t | 동일 |
| 총중량 한도 | ≤ 40t | 동일 |

→ 모듈러 운송은 **폭 3.0m 이상**이라 거의 다 광폭 운송 허가 차량 사용.

## 한국에서 사용되는 트레일러 종류

### ① 일반 평판 트레일러 — 모듈러 부적합 ❌

| 항목 | 40FT (3축) | 45FT |
|------|-----------|------|
| 적재함 길이 | 12.4m (12,380mm) | 13.7m |
| 적재함 폭 | **2.47m** | 2.5m |
| 적재함 높이 | 1.4m | 1.4m |
| 차량 자체중량 | 6.56t | 6.8t |
| 최대 적재 | 24.2t | 25t |
| 차량 총중량 | 30.76t | 31.8t |

> 출처: 두성특장차 평판 트레일러 40FT 3축 (etrucks.co.kr/idx=615)

**폭 2.47m**라 **모듈(폭 3.0~3.4m) 운송 불가**. 일반 화물·작은 패널 운송용.

### ② 저상 트레일러 (Low-bed) — 모듈러 표준 ✅

| 항목 | 수치 |
|------|-----|
| 적재함 길이 | 11~13m (표준 12m) |
| 적재함 폭 | **3.0m** (광폭 운송 허가) |
| 바닥 높이 | 0.4~0.7m (일반 평판 1.4m보다 낮음) |
| 적재 중량 | **24~25t** |
| 축수 | 3축 |
| 차량 자체중량 | 약 12t (트랙터 포함) |

> 출처: 한국특장차 3축 초저상 슬라이드 로우베드 트레일러 (kortrailer.com)

**한국 모듈러 운송에 가장 일반적**. LH 매뉴얼 §3.2.3 단위모듈(폭 3.3m)도 이 차량에 적재.

### ②-2 초저상 트레일러 (Ultra Low-bed) — 높은 모듈용 ✅

| 항목 | 수치 |
|------|-----|
| 적재함 길이 | 12m |
| 적재함 폭 | **3.0m** (광폭 운송 허가) |
| 바닥 높이 | **0.4~0.5m** (일반 저상 0.7m보다 낮음) |
| 적재 중량 | **25t** |
| 축수 | 3축 |

> 출처: 한국특장차 초저상 슬라이드 로우베드 트레일러, 이트럭스 초저상 멀티 트레일러 (etrucks.co.kr)

바닥 높이가 0.4m로 낮아 **높이가 큰 완성 모듈**(층고 3m+)을 도로 높이 한도(4.2~4.5m) 안에서 운송 가능. 일반 저상(0.7m)보다 0.3m 여유 확보.

### ②-3 2축 소형 저상 트레일러 — 좁은 도로 진입용 ✅

| 항목 | 수치 |
|------|-----|
| 적재함 길이 | **9m** |
| 적재함 폭 | **3.0m** (광폭 운송 허가) |
| 바닥 높이 | 0.7m |
| 적재 중량 | **15t** |
| 축수 | 2축 |

> 출처: 동우특장차 LBAAD-01 2축 로우베드 트레일러 (komachine.com)

길이가 짧고 2축이라 **이면도로·좁은 도로 진입**에 유리. 소형 모듈(욕실·현관 등) 운송용.

### ③ 광폭 슬라이드 트레일러 (Extendable Lowbed) — 광폭 모듈용 ✅

| 항목 | 수치 |
|------|-----|
| 적재함 길이 | **16~18m** (슬라이드 확장형) |
| 적재함 폭 | **3.4m** |
| 적재 중량 | **25~28t** |
| 광폭 운송 허가 | 필수 (국토부 ospermit.go.kr 사전 신청) |
| 호송차량 | 1~2대 (3.4m 폭 시 야간 운송 의무) |
| 운행 가능 도로 | **광로 전용** (폭 3.4m가 일반도로 3.3m 초과) |

거실 광폭 모듈(LH 3.6~3.9m), 장축 모듈 운송용.

### ④ A-frame 장착 평판 트레일러 — PC 벽체 패널 ✅

| 항목 | 수치 |
|------|-----|
| 적재함 길이 | 12m |
| 적재함 폭 | 3.0m |
| 적재 중량 | 24t |
| A-frame 받침대 | 평판 위에 임시 장착 (탈착식) |
| 패널 적재 자세 | 세움 (on-edge), 30~60° 각도 |

> 한국에서는 PC 패널 전용 A-frame 트레일러가 별도 모델로는 흔하지 않고, **평판 트레일러 위에 A자형 받침대를 임시 장착**해서 사용. PCI/Wells Concrete 미국 표준과 동일 원리.

## 광폭 운송 허가 절차 (국토부)

1. 국토교통부 제한차량 운행허가시스템 접속 (https://www.ospermit.go.kr/)
2. 운송 경로 (출발지·목적지) + 차량·화물 제원 입력
3. 검토 (3~5일 소요)
4. 허가증 발급 → 호송차량 동반 + 야간 운송 의무 (폭 > 3.0m)

## 한계 및 보완 필요

- 한국특장차·동우특장차 등 모델별 정확한 제원은 **메이커 카탈로그 직접 확인 필요** (kortrailer.com·etrucks.co.kr 사양표 일부 미공개)
- A-frame 평판 사양은 한국 운송사별로 받침대 규격 다름 — **운송 발주 시 견적서 확인**
- 광폭 운송 비용·호송차량비는 운송 거리·도로 등급에 따라 변동
